---
name: Look at screenshots carefully
description: User gets frustrated when I don't match visual details from reference screenshots they provide
type: feedback
---

When the user provides a screenshot as a visual reference, examine every detail carefully before implementing — colors, fills, borders, layout, spacing. Don't approximate or guess. If a button is clearly filled green in the reference, make it filled green, not outlined blue.

**Why:** The user has been burned by me missing obvious visual details in reference screenshots and having to correct me multiple times.

**How to apply:** Before writing component styles, list out every visual property visible in the reference for each element. Match them precisely.
