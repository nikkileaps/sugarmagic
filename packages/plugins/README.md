# `packages/plugins`

Plugin capability model and plugin-host integration.

Owns:

- plugin SDK contracts
- plugin lifecycle hooks
- runtime and shell capability seams
- built-in plugin homes

Does not own:

- hidden mutation paths
- core domain ownership
