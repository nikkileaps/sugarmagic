# Sugarlang

Sugarlang is the adaptive language-learning plugin for Sugarmagic. This directory is the permanent home for the runtime, compile-time, placement, telemetry, UI, data, and test surfaces described in [Proposal 001](./docs/proposals/001-adaptive-language-learning-architecture.md).

The skeleton landed in Epic 1, and later epics fill in the real runtime/compiler/data/editor surfaces. The API documentation index lives at [docs/api/README.md](./docs/api/README.md), and the Studio-side editor contribution module lives under [ui/shell](./ui/shell/README.md).
