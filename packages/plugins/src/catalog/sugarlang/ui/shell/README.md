# Sugarlang Studio Shell

This module owns Sugarlang's Studio-facing editor contributions. These components are editor-only surfaces that sit on top of the runtime/compiler contracts without becoming new sources of truth.

Epic 12 makes this directory the canonical home for:

- NPC inspector Sugarlang role controls
- Scene density and compile-status panels
- Manual rebuild affordances
- Placement questionnaire bank viewing
- Quest-node placement event hints
