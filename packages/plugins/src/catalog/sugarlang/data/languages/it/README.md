# Italian Data

This directory will hold the sugarlang Italian language assets.

Planned sources for the implemented data set:

- OpenSubtitles-derived frequency information
- Kelly project data
- Merged CEFR band assignments for the runtime atlas
- Italian morphology data for lemmatization
- Deterministic simplification mappings
- The plugin-owned placement questionnaire for Italian

Epic 1 leaves all JSON files as placeholders only. Epic 4 will populate and validate the real data.
