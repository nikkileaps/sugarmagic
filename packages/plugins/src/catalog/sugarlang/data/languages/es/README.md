# Spanish Data

This directory will hold the sugarlang Spanish language assets.

Planned sources for the implemented data set:

- ELELex for CEFR-graded lexical coverage
- Spanish morphology data for lemmatization
- Deterministic simplification mappings
- The plugin-owned placement questionnaire for Spanish

Epic 1 leaves all JSON files as placeholders only. Epic 4 will populate and validate the real data.
