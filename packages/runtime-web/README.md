# `packages/runtime-web`

Browser-specific adapters for the shared runtime.

Owns:

- boot adapters
- worker hosts
- transfer helpers
- browser asset/input/save adapters

Does not own:

- alternate runtime semantics
- editor-specific shell assumptions
