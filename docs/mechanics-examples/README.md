# Mechanics Examples

These JSON5 examples are authoring references for humans and LLMs. Studio can
accept JSON5-style input, validates it against
`packages/domain/schemas/mechanics.schema.json`, then persists the project file
as standard JSON.

- `current-caster.mechanics.json5` mirrors the current Sugarmagic caster flow.
- `rackwick.mechanics.json5` shows battery, resonance, and chaos dampening.
- `dnd-5e-attacks.mechanics.json5` shows a non-magic tabletop-style ruleset.
